
# 🗺️ Static Occupancy Grids (Maps)

This directory acts as the robot's "Long-Term Memory". It stores the static environment layouts generated during Phase 2 (SLAM Mapping) so the robot doesn't have to re-explore the same building every time it boots up.

## 🗄️ File Structure Explained

Every saved map consists of a coupled pair of files:

1. **`.pgm` (Portable Gray Map):** The raw 2D image file containing the black (walls), white (free space) and gray (unknown) pixels generated by the LiDAR raytracing.
2. **`.yaml` (Configuration):** The metadata that tells Nav2 how to interpret the image (e.g., origin coordinates, resolution scale like `0.05` meters per pixel and occupied thresholds).
